# 기술 스택 (Tech Stack)

## 핵심 (Core)
- **프레임워크**: React Native (Expo)
- **언어**: TypeScript
- **번들러**: Metro

## 아키텍처 (Architecture)
- **패턴**: Clean Architecture + MVVM
- **라우팅**: React Navigation (Stack, Tab)
- **상태 관리**: Redux Toolkit (전역), React Hooks (로컬/ViewModel)

## 라이브러리 (Libraries)
- **네비게이션 (Navigation)**:
    - `@react-navigation/native`
    - `@react-navigation/stack`
    - `react-native-screens`
    - `react-native-safe-area-context`
- **상태 관리 (State)**:
    - `@reduxjs/toolkit`
    - `react-redux`
- **스타일링 (Styling)**:
    - `StyleSheet` (기본)
